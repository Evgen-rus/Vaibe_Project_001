Суть проекта заключается в разработке и интеграции в чат-интерфейс продукта модуля нейропродажника, 
который будет:
Квалифицировать пользователей в начале триал-периода, задавая им серию вопросов для сбора данных о профиле, 
целях использования продукта и ожидаемых объемах работы.
На основе ответов пользователя предлагать ему персональные тарифные планы.
Отслеживать окончание триал-периода и напоминать пользователю о необходимости покупки.
Удерживать пользователя на триал-версии, если он отказывается от покупки, переводя его в ограниченный 
"войтовый" режим.


ТЗ для MVP "Нейропродажник" (Telegram Bot на aiogram 3.19.0)

Цель
Создать Telegram-бота, который:
Проводит онбординг через серию вопросов
Рекомендует персональный тариф на основе ответов
Управляет триал-периодом (14 дней)
Переводит в ограниченный режим после триала
Стек технологий
Python 3.10+
aiogram 3.19.0 (асинхронный режим)   https://docs.aiogram.dev/en/v3.19.0/
SQLite
OpenAI API gpt-4.1-nano https://platform.openai.com/docs/guides/structured-outputs?api-mode=chat&example=ui-generation

Основные функции
1. Онбординг (FSM)
Приветственное сообщение с кнопкой "Начать тест"

5-7 вопросов о:
Сфере бизнеса
Ожидаемом объеме использования
Бюджете
Ответы сохраняются в БД

2. Подбор тарифа
Анализ ответов через ChatGPT (запрос с контекстом)
Вывод 3 вариантов тарифов с кнопками:
"Выбрать тариф"
"Показать подробности"
"Связаться с менеджером"

3. Триал-период (14 дней)
При старте триала записывается дата окончания
За 1 день бот присылает напоминание:
"Завтра заканчивается триал! Хотите продлить?"
После окончания:
Ограничение функционала
Кнопка "Перейти на платную версию"

4. Админ-панель
Команда /admin с статистикой:
Количество активных пользователей
Конверсия в оплату
Популярные тарифы
